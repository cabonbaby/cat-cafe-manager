貓咪咖啡店店長

這是一個純前端 web 小遊戲。

玩法重點
- 點擊吧台動作生產咖啡與魚麵包
- 滿足排隊貓客人的需求，賺取金幣與人氣
- 購買升級，讓店面更有效率
- 在一天結束前達成今日人氣目標

本機啟動
1. cd /home/suker/projects/cat-cafe-manager
2. python3 -m http.server 4080
3. 打開 http://127.0.0.1:4080

檔案
- index.html：畫面結構
- styles.css：視覺樣式
- app.js：遊戲邏輯
- docs/plans/2026-04-15-cat-cafe-manager.md：規劃文件

之後可用 hermes-demo-kit scaffold quick tunnel 支援。
